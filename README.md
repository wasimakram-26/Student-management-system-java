# Student Management System (Core Java)

## Description
A simple, menu-driven Student Management System developed using Core Java.
The application allows users to perform basic CRUD operations on student
records without using any database.

## Features
- Add new student details
- View all students
- Update student information
- Delete student records
- Console-based interaction

## Concepts Used
- Core Java
- Object-Oriented Programming (OOP)
- Java Collections (ArrayList)
- Exception Handling
- Modular code structure

## How to Run
1. Compile all Java files inside the src folder
2. Run Main.java
3. Follow the menu options in the console

## Note
This project is intentionally implemented without database connectivity
to focus on Java fundamentals and application logic.
